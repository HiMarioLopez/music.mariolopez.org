# backend

AWS Lambda project built with TypeScript.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Testing

```bash
pnpm test
```

## Building

```bash
pnpm build
```

## Deployment

```bash
pnpm cdk:deploy
```

## Project Structure

- `src/handlers/`: Lambda function handlers
- `src/services/`: Business logic
- `src/utils/`: Utility functions
- `src/models/`: Data models
- `tests/`: Test files
